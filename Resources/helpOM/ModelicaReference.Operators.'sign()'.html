Information

Sign function of a Real or Integer number

Syntax

sign(v)

Description

Is expanded into "noEvent(if v > 0 then 1 else if v < 0 then -1 else 0)". Argument v needs to be an Integer or Real expression.

Examples

sign({-3, 0, 3})
 = {-1, 0, 1}
