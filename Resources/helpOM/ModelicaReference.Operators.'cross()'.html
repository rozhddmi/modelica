Return cross product of two vectors

Syntax

cross(x, y)

Description

Returns the cross product of the 3-vectors x and y, i.e.

cross(x,y) = vector( [ x[2]*y[3]-x[3]*y[2];
                       x[3]*y[1]-x[1]*y[3];
                       x[1]*y[2]-x[2]*y[1] ] );
